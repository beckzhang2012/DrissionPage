# DrissionPage 项目运行指南

## 📋 项目简介

DrissionPage 是一个基于 Python 的网页自动化工具，既能控制浏览器，也能收发数据包。

## 🚀 快速开始

### 1. 安装依赖

确保使用正确的 Python 版本（3.6+），然后安装依赖：

```bash
python -m pip install -r requirements.txt
```

**注意**：确保 `python` 命令指向的是您要使用的 Python 版本。如果系统有多个 Python 版本，使用 `python -m pip` 可以确保安装到正确的环境。

### 2. 运行示例

项目已包含一个简单的示例文件 `simple_example.py`，您可以直接运行：

```bash
python simple_example.py
```

这个示例会：
- 启动浏览器
- 访问百度首页
- 在搜索框中输入 "DrissionPage"
- 点击搜索按钮

### 3. 使用项目

#### 方式一：直接使用（推荐用于开发测试）

将项目目录添加到 Python 路径，然后直接导入使用：

```python
import sys
import os
sys.path.insert(0, os.path.dirname(os.path.abspath(__file__)))

from DrissionPage import ChromiumPage

# 创建页面对象
page = ChromiumPage()

# 访问网页
page.get('https://www.baidu.com')

# 查找元素并操作
page.ele('#kw').input('DrissionPage')
page.ele('#su').click()
```

#### 方式二：安装到系统

如果项目完整且所有依赖都正确，可以尝试安装：

```bash
python setup.py install
```

或者开发模式安装：

```bash
pip install -e .
```

**注意**：由于 setup.py 在导入时需要运行项目代码，可能存在循环依赖问题。如果遇到问题，建议使用方式一。

## 📝 基本使用示例

### 示例1：控制浏览器登录

```python
from DrissionPage import ChromiumPage

# 创建页面对象
page = ChromiumPage()

# 访问登录页面
page.get('https://gitee.com/login')

# 输入用户名
page.ele('#user_login').input('您的账号')

# 输入密码
page.ele('#user_password').input('您的密码')

# 点击登录按钮
page.ele('@value=登录').click()
```

### 示例2：获取网页信息

```python
from DrissionPage import ChromiumPage

page = ChromiumPage()
page.get('https://www.example.com')

# 获取页面标题
title = page.title

# 获取页面URL
url = page.url

# 获取页面HTML
html = page.html
```

## 🔧 常见问题

### 问题1：ModuleNotFoundError

**解决方案**：
- 确认使用正确的 Python 版本
- 使用 `python -m pip install` 而不是 `pip install`
- 检查 Python 路径是否正确

### 问题2：无法导入 DrissionPage

**解决方案**：
- 将项目目录添加到 `PYTHONPATH`
- 或者在代码中使用 `sys.path.insert(0, '项目路径')`

### 问题3：浏览器连接失败

**解决方案**：
- 确保已安装 Chromium 内核浏览器
- 检查浏览器路径配置
- 查看项目文档中的浏览器配置说明
